# 🐍 Snake Game using Python

This is a simple Snake Game developed using Python and Pygame.

## 🎮 Features
- Snake movement using arrow keys
- Food system
- Score tracking
- High score saving
- Game Over screen
- Restart option

## 🛠 Technologies Used
- Python
- Pygame

## ▶️ How to Run
1. Install pygame:
   pip install pygame

2. Run the game:
   python snake_game.py
